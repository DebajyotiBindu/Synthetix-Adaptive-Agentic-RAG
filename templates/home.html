Under the Hood: The Hybrid Architecture

To achieve extreme performance, security, and scalability, Synthetix uses a robust, decoupled micro-framework. When you send a query, the system interacts with a multi-layered pipeline working seamlessly behind the scenes.

Django Core & Sessions: Django manages the interface, cookies, session IDs, and safe persistent database storage (sessions and messages). This ensures your conversation thread is safely secured and tracked.

FastAPI Model Calling: The resource-heavy language model and RAG orchestrator are deployed using an isolated, high-speed FastAPI endpoint, plugged perfectly into the Django environment. This isolates calculating tasks from your main web interface, letting the system fetch vector indices, assemble context history, and stream responses at lightning speeds without blocking web workers.

Adaptive Generation: The system extracts the most recent messages, pairs them with relevant document vectors, and feeds the block to the local AI model, which has been strictly system-prompted to drop citation spam and format natural, conversational layouts.
